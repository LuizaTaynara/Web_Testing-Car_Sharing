## 🚗 Web Testing Project – Car Sharing Platform

### Objective
Planning, documentation, and execution of testing activities for a car-sharing web application, focusing on functional validation, usability, responsiveness, and cross-browser compatibility.

---

### Testing Scope
- Validation of the vehicle booking workflow.
- Verification of the booking form interface.
- Testing of payment and card registration functionalities.
- Validation of the vehicle rental process.
- Assessment of layout responsiveness and adaptability.
- Cross-browser and cross-resolution testing.

---

### Test Environments

| Browser | Resolution |
|----------|------------|
| Google Chrome | 800x600 |
| Mozilla Firefox | 1920x1080 |

---

### Tools Used
- Figma
- Google Chrome DevTools
- Firefox Developer Tools
- Jira
- Git/GitHub

---

### Activities Performed
- Analyzed requirements and prototypes available in Figma.
- Created UI testing validation checklists.
- Designed test cases using test design techniques.
- Executed functional and exploratory tests.
- Validated visual components and user interface elements.
- Verified visual consistency across different browsers.
- Performed responsive testing and layout behavior validation across multiple screen resolutions.
- Identified, documented, and reported software defects.
- Recorded test evidence and execution results.
